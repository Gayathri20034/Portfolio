


# Gayathri's Portfolio

A modern, responsive portfolio website built using React, Vite, TypeScript, and Tailwind CSS to showcase projects, skills, and experience in Data Science and Analytics.

## Features

- Clean and modern user interface  
- Fully responsive design  
- Project showcase section  
- Data science and analytics projects  
- Downloadable CV  
- Contact form with Gmail integration  
- Smooth animations using Framer Motion  

---

## Tech Stack

- React  
- TypeScript  
- Vite  
- Tailwind CSS  
- Framer Motion  
- Lucide Icons
